# nodejs例子



## 安装依赖 & 启动

```

//进入nodeexample目录安装依赖
npm install

//启动项目
node app.js

```



## 相关依赖

- ejs:模板引擎选择ejs，我只想要一个简单的帮我填充数据的模板。[github](https://github.com/tj/ejs)
- express:一个服务器端的基于 Node.js 的 JavaScript 开发框架。 [官网](http://expressjs.com/)
- mongoose:让NodeJS更容易操作Mongodb数据库 [官网](http://mongoosejs.com/)
